# Calculadora-de-IMC-JS-puro
Projeto simples de uma calculadora de IMC

## Sobre o projeto

A aplicação permite que o usuário informe sua altura e peso para calcular o IMC e visualizar sua classificação de acordo com a tabela de referência.

## Funcionalidades

- Inserção de altura e peso
- Cálculo automático do IMC
- Exibição da situação atual do usuário
- Tabela com classificações de IMC
- Botão para limpar os campos
- Botão para voltar à tela inicial

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## Como usar

1. Abra o projeto no navegador
2. Informe sua altura  
   Exemplo: `1,75`
3. Informe seu peso  
   Exemplo: `85,5`
4. Clique em **Calcular**
5. Veja o resultado do seu IMC e a classificação correspondente

Este projeto foi desenvolvido com foco em prática de manipulação do DOM, eventos e lógica com JavaScript.
